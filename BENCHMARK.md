# Бенчмарки

## Параметры бенчмарка

- размер данных для построения: `10_000`, `100_000`, `300_000`
- размер данных для поиска: `10_000`, `100_000`, `1_000_000`
- радиус: `0.1`, `1.0`, `10.0`
- `QUERY_COUNT = 500`
- прогрев: `3 x 500 ms`
- измерение: `5 x 500 ms`
