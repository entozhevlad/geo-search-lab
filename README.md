# Геопоиск по Lat/Lng на Quadtree

Минимальная учебная реализация: можно вставлять точки и искать их по близости.

## Что умеет

- хранить точки `(lat, lng)`
- искать точки в прямоугольнике через `searchInBox(...)`
- искать точки по радиусу через `searchInRadius(...)`

## Идея Quadtree

У корня есть прямоугольная область карты:

- `lat in [-90, 90]`
- `lng in [-180, 180]`

Пока точек в узле мало, они просто лежат в списке. Когда лимит превышен, узел делится на 4 части, и точки раскладываются по детям.

Поиск по радиусу работает просто:

1. строится квадрат вокруг точки запроса
2. Quadtree ищет кандидатов в этом квадрате
3. кандидаты фильтруются по обычному евклидову расстоянию
